快速幂 & 逆元

快速幂的原理十分简单

[!danger|iconVisibility:hidden]

快速幂的代码如下

typedef long long ll;
ll qmi(ll a,ll b,ll p)
{
    ll ans = 1;
    while(b)
    {
        if(b & 1) ans = ans * a % p;
        a = a * a % p;
        b >>= 1;
    }
    return ans;
}
