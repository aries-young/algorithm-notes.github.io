快速幂 & 逆元

快速幂的原理十分简单 ak=a20×a21×a22×…×a2x, k=20+21+22+…+2x a^k=a^{2^0}\times a^{2^1}\times a^{2^2}\times\ldots\times a^{2^x},\ k=2^0+2^1+2^2+\ldots+2^x a​k​​=a​2​0​​​​×a​2​1​​​​×a​2​2​​​​×…×a​2​x​​​​, k=2​0​​+2​1​​+2​2​​+…+2​x​​

快速幂的代码如下

typedef long long ll;
ll qmi(ll a,ll b,ll p)
{
    ll ans = 1;
    while(b)
    {
        if(b & 1) ans = ans * a % p;
        a = a * a % p;
        b >>= 1;
    }
    return ans;
}

我们先说一下逆元的概念

当 a⋅m≡1 (mod p)a\cdot m \equiv 1\ (\text{mod}\ p)a⋅m≡1 (mod p)， mmm 即为 aaa 在 mod p\text{mod}\ pmod p 意义下的逆元（需要注意只有 aaa 和 ppp 互质，aaa 才有关于 ppp 的逆元）

在模运算中，没有 (a/b) mod p=((a mod p)/(b mod p)) mod p(a/b)\ \text{mod}\ p=\left((a\ \text{mod}\ p) / (b\ \text{mod}\ p)\right)\ \text{mod}\ p(a/b) mod p=((a mod p)/(b mod p)) mod p 这回事， 所以通过乘法逆元帮助我们将其转换为乘法形式，即 (a/b) mod p=(a⋅x) mod p(a/b)\ \text{mod}\ p=(a\cdot x)\ \text{mod}\ p(a/b) mod p=(a⋅x) mod p，其中 xxx 表示 bbb 的逆元

既然有如此好事可以化除为乘，那怎么求 xxx 呢？

首先 xxx 作为 bbb 的逆元，有 b⋅x (mod p)≡1 b\cdot x\ (\text{mod}\ p) \equiv 1 b⋅x (mod p)≡1 又由于 b,pb, pb,p 互为质数，由费马小定理得 b(p−1)=1 (mod p) b^{(p-1)}=1\ (\text{mod}\ p) b​(p−1)​​=1 (mod p) 于是有 x=b(p−2) x=b^{(p-2)} x=b​(p−2)​​
